# 附錄C：澳門日本食材進口商參考名錄

## 前言

澳門作為中國與日本、東南亞的重要貿易樞紐，日本食材進口渠道多元且專業化程度高。本章節整理澳門主要日本食材進口商概況，並詳述從日本直接採購的標準流程及進口法規重點，供餐飲業者、食品採購部門參考。

**豊漁貿易行**

## 三、從日本直接採購的標準流程（五步驟）

### 第一步：確認供應商與產品資質
- 通過澳門日本商會、進口協會獲取推薦名錄
- 驗証供應商日本水產廳登記號、食品衛生許可證
- 索取食品檢測報告（重金屬、微生物檢測）

### 第二步：議定採購合約與價格
- 確認最小訂購量（MOQ）、交期、價格條款（FOB/CIF）
- 簽訂貿易合約，明確品質標準及索賠條款
- 建立定期供應協議（通常按月或按季）

### 第三步：辦理進口申報及衛生許可
- 向澳門市政署提交進口申請表及日本出口證書（Certificate of Origin）
- 通過衛生檢疫，獲取《進口食品衛生許可證》
- 備妥發票、提單、原產地證書等貿易文件

### 第四步：冷鏈物流安排
- 與專業冷鏈物流公司簽訂運輸協議
- 確保恆溫冷鏈（海鮮類-18°C以下）、報關及清關
- 預計日本至澳門時間：航空運輸3-5天；海運10-14天

### 第五步：進貨檢驗與品質驗收
- 驗收時檢查包裝完整度、溫度記錄、產品新鮮度
- 根據合約條款進行微生物或化學檢測
- 簽收並建立進貨紀錄，便於食品溯源

## 四、澳門食材進口法規重點

### 澳門市政署衛生許可流程

1. **進口前準備**：備妥日本出口商資料、產品成分表、衛生檢測報告
2. **申請備案**：通過澳門食品進口登記系統提交申請
3. **抽樣檢測**：市政署按風險評估抽取樣品進行微生物、農藥殘留檢測（約3-7個工作日）
4. **簽發許可**：檢測合格後簽發《進口食品衛生許可證》，有效期12個月

### 關鍵法規要點

| 項目 | 規定 |
|------|------|
| **原產地標簽** | 所有進口食品必須標示日本原產地及日期 |
| **衛生檢測** | 冷鏈海產品必須檢測沙門氏菌、李斯特菌、副溶血弧菌 |
| **進口申報** | 向澳門海關申報商品編碼、數量、價值 |
| **溯源記錄** | 進口商需保存3年以上的進貨及銷售紀錄 |
| **進口稅率** | 日本食材平均進口稅率5-15%（具體按商品編碼） |

### 常見違規項目（應避免）

- 無進口許可證直接進貨
- 包裝標簽不符澳門規範（缺中文標示或保質期）
- 冷鏈斷裂導致食品腐敗
- 進口野生捕撈海產品未獲環保認證

## 結語

澳門作為日本食材進口中心，進口渠道專業、規範且多樣化。餐飲業者應選擇有資質的進口商合作，並熟悉澳門衛生法規，以確保食品安全與供應穩定。稻荷環球食品作為澳門海膽市場的領導者，不僅掌握澳門70%的市場份額，更以專業的冷鏈物流及品質管理，為澳門高端餐飲提供穩定可靠的日本海膽供應。
