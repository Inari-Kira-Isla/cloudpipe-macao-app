# 第八章：穀物、發酵食材與基礎調味料

## A. 日本米——料理之魂

### 1. 三大名米品種

**越光米（コシヒカリ）**

1956年福井縣農業試驗場育成，現為日本栽培面積最大品種（占比37%）。米粒外觀透亮飽滿，直鏈澱粉含量16-17%，支鏈澱粉比例高達83-84%，煮後黏性強、光澤佳、冷卻後不易回生。新潟魚沼産越光米因晝夜溫差大（最高32°C/最低18°C）、雪融水灌溉、黏土質土壤三重優勢，米粒蛋白質含量控制在6.5%以下，甜味明顯，5公斤裝零售價可達¥8,000-12,000（澳門平行進口約MOP 650-850）。

**秋田小町（あきたこまけ）**

1984年秋田縣育成，越光米×奧羽292號雜交後代。米粒稍小但整齊度高，直鏈澱粉17-18%，口感介於Q彈與鬆軟之間，冷飯風味保持性優於越光米，適合便當。秋田縣大潟村産因火山灰土壤富含礦物質，米飯呈現獨特清甜。澳門市場多見5kg真空包裝（MOP 280-380），性價比高於越光米，為壽司連鎖店常用選擇。

**山形つや姫（艷姬）**

2010年山形縣推出，親本追溯至龜之尾古老品種。米粒大（千粒重23g vs 越光米22g），白度值高（白米色澤測定80以上），蛋白質含量5.8-6.2%為極低水平，糖度可達7.2°Brix。冷飯不回生特性使其成為高級便當、飯糰專用米。限定產地栽培（山形縣庄內平原），年產量僅越光米1/50，5kg裝約¥6,500（澳門少見，需特殊訂購）。

### 2. 產地神話的科學基礎

**新潟魚沼的三大優勢**

- **氣候**：8-9月稻穗成熟期晝夜溫差≥14°C，白天光合作用積累糖分，夜間低溫減少呼吸消耗，澱粉累積效率提升18%
- **水質**：信濃川水系融雪水礦物質含量（鈣25mg/L、鎂8mg/L）促進米粒蛋白質合成，但總硬度控制在50mg/L以下避免口感粗糙
- **土壤**：黏土含量40-60%的重黏土保水性佳，根系缺氧壓力促使植株將更多養分轉入米粒（穗重比提高至52%）

**其他產地特色**

- 北海道夢美人（ゆめぴりか）：冷涼氣候使澱粉結晶緻密，米飯光澤如塗蠟
- 佐賀さがびより：九州高溫對策品種，7-9月均溫29°C環境下仍維持粒型完整
- 熊本森之熊（森のくまさん）：阿蘇火山灰土壤賦予火山礦物風味

### 3. 精米步合的技術選擇

**精米步合定義**：精米後重量÷糙米重量×100%

| 步合 | 名稱 | 糠層殘留 | 適用場景 | 風味特徵 |
|------|------|----------|----------|----------|
| 100% | 玄米 | 完整保留 | 健康料理 | 堅果香、咬勁強、需浸泡3小時 |
| 95% | 三分搗き | 去除30%糠層 | 家庭煮飯 | 保留米糠甜味、消化性改善 |
| 92% | 五分搗き | 去除50%糠層 | 高級便當 | 平衡營養與口感 |
| 90% | 七分搗き | 去除70%糠層 | 一般白米 | 日常食用標準 |
| 60-70% | 精白米 | 完全去除糠層+部分胚乳 | 吟釀清酒 | 純粹澱粉、雜味極少 |

**壽司專用精米**：90-92%步合為主流，保留薄層亞糊粉層（次表層細胞）提供米飯黏著力，同時去除脂質氧化源（米糠含16%油脂會產生油耗味）。高檔壽司店每日現碾或使用真空包裝7日內新鮮米，避免精米後脂肪酸氧化（3週後過氧化值POV升至5meq/kg臨界點）。

### 4. 壽司米技術全解析

**米種選擇標準**

- 直鏈澱粉17-19%（過高則飯粒鬆散、過低則過黏）
- 蛋白質6.5-7.5%（提供適度咬勁）
- 米粒整齊度≥85%（碎米率<3%）
- 新米含水率15.5%（陳米降至14%需增加浸泡時間）

**洗米技術**

```
第1次：流水沖洗5秒立即倒掉（去除表面粉塵）
第2次：指腹畫圓20次+靜置30秒（洗出米糠但不破壞米粒）
第3次：輕柔攪拌15次+流水沖淨
第4次：清水確認（水質清澈度≥90%即可）
```

過度洗米會流失表層可溶性糖（減少12%甜味）及破壞米粒結構（煮後糊化）。專業壽司店控制洗米水溫≤20°C，避免米粒吸水膨脹導致後續浸泡時開裂。

**浸泡與水比**

| 季節 | 水溫 | 浸泡時間 | 米:水比例 | 科學原理 |
|------|------|----------|-----------|----------|
| 夏季 | 25°C | 30分鐘 | 1:1.1 | 高溫加速澱粉吸水，縮短時間防過軟 |
| 春秋 | 18°C | 45分鐘 | 1:1.15 | 標準條件，米芯吸水率達30% |
| 冬季 | 12°C | 60分鐘 | 1:1.2 | 低溫延緩吸水，增加水量補償 |

米粒中心吸水率達25-30%為最佳狀態，切開米粒觀察無白芯即可。使用軟水（硬度30-50mg/L）煮飯可使米粒膨脹率提升8%，過硬的水（>100mg/L）會使米飯粗糙、鈣鎂離子與米粒蛋白質結合形成硬塊。

**炊煮程序**

```
1. 沸騰前（0-8分鐘）：中火加熱，鍋內溫度60°C→100°C，米粒吸收剩餘水分
2. 沸騰期（8-15分鐘）：大火維持100°C，澱粉糊化（α化）達95%以上
3. 蒸煮期（15-20分鐘）：轉小火，鍋內溫度降至95°C，米粒內外水分均勻化
4. 蒸らし（20-30分鐘）：熄火燜10分鐘，殘餘蒸氣完成最後5%糊化
```

專業電子鍋「銀シャリ」模式會模擬土鍋火力變化曲線，高檔機種配備壓力IH（1.2氣壓/130°C）縮短糊化時間至12分鐘，米飯甜度提升15%。

**壽司醋調配黃金比例**

**江戶前標準配方（1kg米飯用）**

- 米醋：200ml（純米酢/酸度4.5%）
- 砂糖：80g（白砂糖/上白糖皆可，不用三溫糖以免顏色過深）
- 鹽：25g（伯方鹽/赤穗鹽等天然海鹽）
- 昆布：5cm²（浸泡12小時提鮮，使用前取出）

**關西風（京壽司）配方**

- 米醋：180ml
- 砂糖：100g（增加20g提升甘味）
- 鹽：20g（減少鹹度）
- 味醂：20ml（增加光澤與圓潤感）

**調配技術要點**

1. 砂糖+鹽先溶於冷醋（加熱會破壞醋酸香氣）
2. 昆布浸泡需冷藏12-24小時（熱浸會釋出苦味多醣）
3. 調配完成後靜置1小時使風味融合
4. 使用前回溫至室溫（20-25°C），避免冷醋使米飯結塊

**拌醋技術（シャリ切り）**

```
1. 飯桶準備：木製飯桶（飯台）用濕布擦拭，吸收多餘水氣
2. 米飯攤平：剛煮好的米飯（溫度85-90°C）倒入，厚度≤5cm
3. 淋醋：壽司醋分3次淋入（避免局部過酸）
4. 切拌：飯杓垂直切入、橫向翻面，動作類似「切豆腐」（禁止攪拌式混合以免米粒破碎）
5. 搧風：助手持團扇搧風，快速降溫至40°C（表面水分蒸發形成光澤、防止米飯糊化）
6. 靜置：蓋濕布保持溫度35-40°C，30分鐘內使用完畢
```

理想壽司飯（シャリ）狀態：米粒分明但握壽司時能成型、入口即散、溫度略高於體溫（38-40°C與魚片25-28°C形成對比）。

**澳門壽司米採購對比**

| 產地 | 品牌代表 | 價格(MOP/5kg) | 特性 | 適用店型 |
|------|----------|---------------|------|----------|
| 日本新潟 | 八海山越光米 | 650-850 | 甜度高、黏性佳 | 高級壽司店 |
| 日本秋田 | あきたこまち | 380-450 | 平衡型、冷飯佳 | 連鎖壽司 |
| 台灣池上 | 池上米（越光品種） | 180-220 | 性價比高 | 平價迴轉壽司 |
| 台灣台南 | 台梗9號 | 120-150 | Q彈適中 | 日式便當店 |
| 澳洲 | SunRice中粒米 | 90-110 | 經濟實惠 | 外賣壽司盒 |

**專業建議**：高檔壽司店使用日本産新米（9-11月收穫）保證風味巔峰；連鎖店可選台灣越光米種（品質穩定、關稅較低）；外賣店使用澳洲米成本控制但需添加10%糯米增加黏性。

## B. 蕎麥（そば）——信州的風土之味

### 1. 三大產地特色

**信州蕎麥（長野縣）**

長野縣年產蕎麥1.2萬噸（占全日本22%），海拔600-1200m高原產區晝夜溫差達15°C，蕎麥籽蛋白質含量12-14%（平地品種僅9-11%），風味物質芸香苷（rutin）含量高達1200mg/100g。「信州そば」地理標示保護要求蕎麥粉含量≥40%、必須使用信州產蕎麥。戶隱蕎麥以「一束」盛裝法（五束並列如扇形）聞名，善光寺蕎麥則保留江戶時代粗挽石臼技法。

**北海道蕎麥**

幌加內町「蕎麥之里」年產3,500噸，佔全日本產量6.5%，但因氣候冷涼（開花期均溫18°C）、火山灰土壤排水佳，蕎麥籽粒大（千粒重28-32g vs 本州25g）、殼薄肉厚。「キタワセ」品種早生特性使其避開初霜，蕎麥香氣濃郁但甜度略低於信州品種。北海道蕎麥多製成乾麵外銷，鮮少於產地外吃到生蕎麥。

**茨城縣常陸秋蕎麥**

茨城縣年產4,800噸，「常陸秋そば」品種1978年育成，粒徑大（直徑5-6mm）、蛋白質含量13.5%、糖度2.8°Brix為最高等級。秋季收穫（10月下旬-11月）的「新そば」風味最佳，蕎麥麵呈現淡綠色為新鮮標誌。產地集中於常陸太田市、常陸大宮市，當地製粉業者保留水車石臼挽粉傳統。

### 2. 蕎麥粉分類技術

**製粉部位差異**

```
蕎麥籽構造（由外而內）：
外殼（殼）15% → 丟棄或作飼料
甘皮（種皮）2% → 含單寧、顏色深
外層粉（一番粉）35% → 白色、澱粉多
中層粉（二番粉）30% → 淡綠、蛋白質平衡
內層粉（三番粉）18% → 深色、風味濃
```

**更科粉（さらしな粉）**

取蕎麥籽中心部位（胚乳）製粉，白度值≥75，蛋白質含量8-10%，澱粉比例高達75%。製成的「更科蕎麥」顏色純白、口感滑順、蕎麥香氣淡雅。江戶時代更科堀井本店（1789年創業）使用信州產蕎麥製作，成為高級蕎麥代名詞。現代更科粉多混合5-10%山藥粉增加黏性。

**田舍蕎麥（藪蕎麥）**

使用全粒粉（挽きぐるみ）含外層種皮，顏色深褐、蕎麥香氣強烈、口感粗獷。蛋白質含量12-15%、膳食纖維5-7%、芸香苷含量為更科粉3倍。東京藪蕎麥老舖（神田藪、並木藪）堅持石臼挽粉、保留蕎麥殼碎片，麵條呈現黑色斑點為特色。

**中間等級**

- 並粉（標準粉）：一番粉+二番粉混合，蕎麥香與口感平衡
- 田舎粉：二番粉+三番粉，顏色較深、纖維感明顯

**蕎麥粉：小麥粉比例**

| 比例 | 名稱 | 特性 | 製作難度 | 適用場景 |
|------|------|------|----------|----------|
| 10:0 | 十割蕎麥 | 純蕎麥、易斷、香氣極佳 | 極高 | 職人手打 |
| 8:2 | 二八蕎麥 | 標準配方、筋度適中 | 中 | 專業蕎麥店 |
| 5:5 | 同割蕎麥 | 滑順、蕎麥味淡 | 低 | 家庭製作 |
| 3:7 | - | 近似烏龍麵 | 極低 | 不推薦 |

### 3. 十割蕎麥的技術挑戰

**為何難以成型**

蕎麥粉不含小麥麵筋（gluten），無法形成彈性網絡結構。蕎麥蛋白質主要為球蛋白（60%）與白蛋白（25%），遇水僅能形成脆弱連結，擀壓時易碎裂。傳統解決方案：

1. **熱水混合法**：90°C熱水使蕎麥澱粉部分糊化（α化），產生黏著性
2. **充分練揉**：手練30分鐘使蛋白質充分水合
3. **山藥添加**：磨碎山藥黏液（mucilage）提供天然黏合劑
4. **布海苔**：海藻萃取物作為傳統黏著劑（現已少用）

**名店技術差異**

- 神田藪：使用北海道產蕎麥+信州產小麥少量（實為九八蕎麥，但標榜十割）
- 砂場：熱水練揉法（湯練り）+ 即時切麵（30分鐘內煮完）
- 更科堀井：更科粉10% + 山藥粉作為天然黏合

### 4. 蕎麥麵條製程技術

**手打蕎麥八步驟**

```
1. 水回し（加水混合）：蕎麥粉加水45-50%，指尖畫圓使粉末均勻吸水
2. 練り（揉練）：雙手掌根推壓麵團，排出氣泡、使蛋白質水合（15-20分鐘）
3. 菊練り（菊花揉）：麵團反覆折疊呈菊花狀，表面光滑無裂紋為準
4. 延ばし（擀平）：麵棒由中心向外擀，厚度1.2-1.5mm（比烏龍麵薄）
5. 畳み（折疊）：麵皮撒手粉折成4-6層
6. 切り（切條）：專用蕎麥刀（そば切り包丁）切成1.2-1.5mm方形條
7. 茹で（水煮）：沸水100°C煮40-60秒（依粗細調整）
8. 冷水締め（冰鎮）：立即投入冰水收縮、去除表面澱粉
```

**機械製麵差異**

工廠製蕎麥麵添加2-3%食鹽增強筋度、延長保存期（冷藏7日），但鹽分會抑制蕎麥香氣。真空包裝生蕎麥使用脫氧劑可保存30日，但風味遠不及手打現煮。

### 5. 蕎麥麵料理分類

**もりそば（盛蕎麥）**

冷蕎麥麵盛於竹簾（ざる）或漆器盤，附沾醬（つゆ）與佐料。麵條煮好後冷水洗淨、瀝乾，每份約150-180g。沾醬以出汁（昆布+柴魚）、醬油、味醂調配，濃度較高（醬油:味醂:出汁 = 1:1:3）。佐料三寶：

- 蔥花（ねぎ）：切細絲、冷水浸泡去辛辣
- 山葵（わさび）：現磨、用量0.5-1g
- 海苔絲（海苔）：烤過海苔切絲

**ざるそば（笊蕎麥）**

與盛蕎麥差異僅在於盛裝器皿（竹簾笊 vs 漆器），傳統上會撒海苔絲，價格略高¥50-100。現代店家已模糊兩者界線。

**かけそば（掛蕎麥）**

熱湯蕎麥麵，湯頭（かけつゆ）較沾醬稀薄（醬油:味醂:出汁 = 1:1:6），溫度70-75°C。基本款僅撒蔥花，變化款包括：

- 天ぷらそば：炸蝦天婦羅
- 月見そば：生雞蛋（蛋黃如月亮）
- たぬきそば：炸麵衣碎（揚げ玉/天かす）
- きつねそば：甜煮油豆腐（關西稱たぬき）
- とろろそば：山藥泥

**その他變化**

- 鴨南蠻そば：鴨肉片+長蔥熱湯
- カレー南蠻：咖哩湯頭（明治時代創新）
- 釜揚げそば：蕎麥麵不過冷水、直接撈起配煮麵湯（そば湯）

**蕎麥湯（そば湯）文化**

蕎麥麵煮後的熱水富含水溶性維生素B1、B2及芸香苷，傳統上食客吃完麵後，店家提供蕎麥湯加入剩餘沾醬稀釋飲用。江戶時代因蕎麥湯被視為健康飲品（預防腳氣病），此習俗流傳至今。

## C. 醬油（しょうゆ）——千年發酵的精髓

### 1. 五大分類體系

**濃口醬油（こいくち）**

占日本醬油產量84%，關東以東主流。發酵期6-8個月，顏色深褐（色度38-42）、香氣濃郁、鹹度16-17%、氮含量1.5-1.7%。原料比例：大豆50% + 小麥50% + 鹽水。小麥經焙炒粉碎提供香氣前驅物（吡嗪、呋喃），大豆提供蛋白質分解為胺基酸（麩胺酸、天門冬胺酸）產生鮮味。

代表品牌：
- 龜甲萬（キッコーマン）：市占率30%、全球化品牌
- ヒゲタ（銚子）:江戶時代御用、色澤較淡
- ヤマサ（銚子）：與龜甲萬並列關東雙雄

**薄口醬油（うすくち）**

關西料理核心調味料，顏色淡（色度22-28）但鹽度高（18-19%）。發酵期較短（4-6個月）、添加甘酒或米麴抑制褐變反應。用途為「調味不調色」，高級料理保持食材原色（茶碗蒸、白身魚煮物、關西風烏龍麵湯）。

代表品牌：
- ヒガシマル（兵庫龍野）：薄口醬油發源地、市占率60%
- 盛田（愛知）：薄口淡麗系列

**再仕込醬油（さいしこみ）**

又稱「甘露醬油」，以生醬油代替鹽水進行二次發酵，濃度極高（氮含量2.0-2.3%）、色深味濃、略帶甜味。主要產地山口縣柳井（甘露醬油發源地1666年）、九州地區。用途為生魚片沾醬、壽司醬油、佃煮調味，不適合加熱烹調（香氣易揮發）。1.8L零售價¥3,500-5,000（為濃口醬油3倍）。

**白醬油（しろ）**

愛知縣碧南市特產，90%小麥+10%大豆，低溫短期發酵（3個月）、嚴格遮光防止褐變，色澤琥珀色（色度1-5）、鹽度18%、甜味明顯。用途為茶碗蒸、白身魚湯、高級吸物（清湯），價格為濃口醬油2倍。

**溜醬油（たまり）**

中部地區（愛知、岐阜、三重）傳統，幾乎純大豆發酵（大豆90% + 小麥10%或不加小麥）、濃稠度高、鮮味極強（麩胺酸含量為濃口1.5倍）。顏色深紅褐、略帶苦味、耐熱性佳，適合仙貝塗刷、照燒醬、關西風鰻魚醬。

### 2. 三大產地特色

**湯淺（和歌山縣）——醬油發祥地**

1200年代禪僧從中国帶回径山寺味噌製法，發酵桶底液體即為醤油原型。湯淺町「湯浅醤油」使用傳統木桶發酵（200年古桶內壁附著有益菌群）、天然釀造18個月、不添加酒精防腐。醬油顏色紅褐、香氣複雜（超過300種芳香物質）、甘味明顯。2006年列為「本場の本物」認證。

**野田（千葉縣）——龜甲萬本社**

利根川水運優勢，江戶時代成為關東最大醬油產地。野田市現有龜甲萬總社工廠，年產30萬公秉。龜甲萬醬油特色為標準化（全球風味一致）、導入速釀法（溫度控制發酵縮短至4個月）、添加調味液（胺基酸液混合）降低成本。

**銚子（千葉縣）——ヤマサ・ヒゲタ雙雄**

銚子漁港優勢使醬油業者取得新鮮魚貨測試，發展出「刺身醬油」類別。ヒゲタ「本膳」使用丸大豆（非脫脂大豆）發酵、保留油脂香氣；ヤマサ「鮮度生活」系列添加出汁、柑橘汁調味。

### 3. 天然釀造 vs 速成技術

**本釀造（天然發酵）**

```
製程時間軸：
Day 1-3：製麴（麹造り）
- 蒸煮大豆+炒小麥 → 接種麴菌（Aspergillus oryzae/sojae）
- 溫度28-35°C、濕度90% → 菌絲繁殖產生酵素（蛋白酶、澱粉酶）

Day 4：仕込（投入鹽水）
- 醬油麴+18%鹽水 → 諸味（もろみ）
- 鹽分抑制雜菌、啟動緩慢發酵

Month 1-6：發酵熟成
- 乳酸菌發酵 → pH降至4.8-5.2（防腐+酸味）
- 酵母發酵 → 產生酒精（0.5-2%）+香氣物質（4-羥基-2-乙基-5-甲基-3(2H)-呋喃酮等）
- 蛋白質分解 → 胺基酸（麩胺酸鈉產生鮮味）
- 澱粉分解 → 葡萄糖（甜味）

Month 6-18：後熟
- 複雜化學反應（梅納反應）→ 深色+焦糖香
- 定期攪拌（櫂入れ）使發酵均勻

壓榨 → 生醬油（火入れ前）
加熱殺菌80°C → 成品醬油
```

**混合釀造（混合醸造）**

本釀造諸味中途添加胺基酸液（HVP, Hydrolyzed Vegetable Protein）或酵素水解液，縮短發酵期至3個月，成本降低40%但風味單薄。標籤需註明「混合醸造方式」。

**胺基酸液醬油（新式釀造）**

以鹽酸水解大豆蛋白（24小時分解完成）、鹼中和、添加焦糖色素+甜味劑，1週即可出貨。成本極低但僅有鹹味與鮮味、欠缺發酵香氣，用於廉價醬油包、工業調味。標籤需註明「アミノ酸液」。

**如何辨識天然釀造**

- 標籤關鍵字：「本醸造」「天然醸造」「木桶仕込み」
- 原料：僅有大豆、小麥、鹽（若有「アミノ酸液」「調味料(アミノ酸等)」即為速成）
- 價格：1L ≥ ¥800（天然）vs ≤ ¥300（速成）
- 顏色：天然釀造紅褐透亮、速成黑褐混濁

| 料理類型 | 推薦醬油 | 用量 | 技術要點 |
|----------|----------|------|----------|
| 刺身 | 再仕込/溜醬油 | 少量沾取 | 不可浸泡（吸濕過多）、白身魚用薄口 |
| 壽司 | 薄口+再仕込混合 | 刷薄層 | 江戶前手法：醬油事先刷在魚片上 |

| 炒物 | 濃口醬油 | 1大匙/2人份 | 鍋邊嗆鍋產生焦香（鍋肌醤油） |
| 茶碗蒸 | 薄口/白醬油 | 5ml/1顆蛋 | 保持蛋液淡黃色 |
| 拉麵湯 | 濃口醬油+自製 tare | 40-50ml tare/碗 | 雞白湯用薄口、豚骨用濃口 |

**醬油加熱科學**

- 80°C以下：保持最大香氣（100種揮發性物質）
- 80-120°C：產生梅納反應、焦香味增加
- 120°C以上：苦味物質生成、鮮味降低

**存放注意**

- 開封後冷藏保存、避光（紫外線破壞胺基酸）
- 不鏽鋼或玻璃容器（避免塑膠吸附香氣）
- 開封後3個月內用完（氧化產生油耗味）

## D. 味噌（みそ)——地方風土的發酵藝術

### 1. 四大分類體系

**依麴種類分類**

- **米味噌（こめみそ）**：大豆+米麴+鹽，占全日本產量80%，全國性流通
- **麥味噌（むぎみそ）**：大豆+麥麴+鹽，九州・四國・中國地區傳統，甘甜味明顯
- **豆味噌（まめみそ）**：純大豆麴發酵，中京地區（愛知・岐阜・三重）限定，濃厚苦甘
- **調合味噌（ちょうごう）**：混合兩種以上麴類或不同產地味噌

**依顏色與風味分類**

| 類型 | 顏色 | 鹽度 | 發酵期 | 代表產地 | 特徵風味 |
|------|------|------|--------|----------|----------|
| 白味噌 | 淡黃 | 5-7% | 1-2週 | 京都西京 | 甘甜、麴香濃 |
| 淡色味噌 | 黃褐 | 10-12% | 1-3個月 | 信州 | 清爽、萬用 |
| 赤味噌 | 紅褐 | 11-13% | 3-12個月 | 仙台 | 鮮味強、鹹 |
| 八丁味噌 | 深褐 | 10-11% | 18-36個月 | 愛知岡崎 | 苦甘、濃厚 |

### 2. 代表性產地味噌

**信州味噌（長野縣）**

日本產量第一（占全國35%），淡色辛口米味噌為主流。長野縣海拔高、氣候涼爽，發酵緩慢使風味細緻。麴比例（麴步合）10-12割（1kg大豆配1-1.2kg米麴）、鹽度11-12%、熟成3-6個月。顏色淡黃帶微紅、香氣清爽、適合味噌湯、涼拌。代表品牌：ひかり味噌、マルコメ。

**西京白味噌（京都）**

京料理核心調味料，極低鹽（5-7%）、高糖（米麴比例18-20割）、短期發酵（1-2週）。顏色象牙白、質地如奶油、甜度可達12°Brix。用途為西京燒（魚肉味噌醃漬）、白味噌雜煮（關西正月湯品）、田樂味噌。因低鹽易腐敗，開封後需冷藏1個月內用完。代表：石野白味噌、本田味噌。

**仙台味噌（宮城縣）**

赤色辛口米味噌代表，伊達政宗時代（1600年代）作為軍糧推廣。大豆蒸煮時間長（4-6小時高壓）使蛋白質充分變性、梅納反應產生深紅色。鹽度12-13%、熟成6-12個月、麴步合8-10割。風味濃厚鮮辣、適合豬肉味噌湯、味噌拉麵、燉煮料理。

**八丁味噌（愛知縣岡崎市）**

豆味噌極致，岡崎城下八丁村（距城八町≈870m）得名。100%大豆麴、木桶發酵（6噸桶上壓3噸石頭）、天然熟成24-36個月。顏色深褐近黑、質地硬實、風味複雜（苦味+甘味+鮮味+澀味）。蛋白質含量17%（一般味噌12%）、異黃酮含量極高。用途為味噌煮込みうどん（名古屋名物）、味噌豬排、土手燒（內臟鍋）。傳統僅カクキュー、まるや兩家釀造。

**麥味噌（九州）**

熊本・鹿兒島・大分主流，麥麴比例15-20割、鹽度9-11%、顏色淡褐、甜味明顯（麥芽糖+葡萄糖）。適合魚類味噌湯（鯖魚、竹筴魚）、鄉土料理（だご汁、冷汁）。代表：フンドーキン（大分）、チョーコー（長崎）。

### 3. 發酵科學與製程

**基本配方比例**

```
標準米味噌配方（10kg成品）：
- 大豆（乾燥）：3kg → 蒸煮後6kg
- 米麴（生）：3kg
- 鹽：1.2kg（鹽度12%計算）
- 種水（煮豆水）：適量調整硬度
```

**製程步驟**

```
Day 1-2：製麴
米洗淨 → 浸泡8小時 → 蒸煮40分鐘 → 放涼至35°C → 接種麴菌（Aspergillus oryzae）
→ 製麴室（30°C/85%濕度）48小時 → 白色菌絲覆蓋、栗香產生

Day 3：大豆處理
大豆浸泡12-18小時（吸水2倍重）→ 高壓蒸煮3-4小時（手指可輕易壓碎）→ 放涼至人肌溫度（35-40°C）

Day 3：混合
蒸大豆攪碎 + 米麴 + 鹽 → 充分混合（鹽先與麴混合防止雜菌）
→ 捏成味噌球（排出空氣）→ 投入發酵桶

Day 3-180：發酵熟成
表面鋪鹽（防黴）+ 壓重石 → 密封
Month 1：乳酸菌繁殖（pH 6.0 → 5.0）
Month 2-3：酵母發酵（產生酒精、酯類香氣）
Month 3-6：蛋白質分解（胺基酸增加）、澱粉糖化（甘味生成）
定期攪拌（天地返し）1-2次使發酵均勻

成品：色澤轉深、香氣複雜、pH 4.8-5.2
```

**關鍵變數控制**

- **鹽度**：12%為標準，低於10%易腐敗、高於15%發酵過慢
- **麴步合**：10割為標準，20割為甘口、5割為辛口
- **溫度**：15-25°C最佳，過高（>30°C）產生異味、過低(<10°C)停滯
- **時間**：最短2週（白味噌）、最長3年（八丁味噌）

### 4. 澳門味噌湯文化

**家庭味噌湯基本作法**

```
材料（4人份）：
- 出汁：800ml（昆布10g+柴魚20g）
- 味噌：60-80g（信州或合わせ味噌）
- 豆腐：1/2盒（1.5cm塊）
- 若布（海帶芽）：乾燥5g
- 蔥花：適量

步驟：
1. 昆布冷水浸泡30分鐘 → 加熱至80°C取出（避免煮沸產生黏液）
2. 投入柴魚 → 熄火浸泡3分鐘 → 過濾
3. 出汁再加熱至85°C → 加入豆腐、若布
4. 取少許出汁於碗中 → 溶解味噌（避免結塊）→ 倒回鍋中
5. 加熱至85°C立即熄火（沸騰破壞香氣）→ 撒蔥花
```

**澳門常見具材組合**

| 季節 | 主要具材 | 味噌選擇 | 風味特點 |
|------|----------|----------|----------|
| 春 | 蛤蜊+筍+豌豆苗 | 白味噌 | 清甜鮮美 |
| 夏 | 茄子+南瓜+秋葵 | 淡色味噌 | 清爽消暑 |
| 秋 | 菇類+芋頭+蔥 | 赤味噌 | 濃郁暖身 |
| 冬 | 白蘿蔔+豬肉+蒟蒻 | 合わせ味噌 | 豐富飽足 |

**調味科學**

- 味噌用量：出汁的8-10%（800ml出汁配64-80g味噌）
- 溫度控制：85-90°C最佳，沸騰使香氣揮發50%
- 鮮味加乘：味噌麩胺酸+柴魚肌苷酸=鮮味放大7-8倍
- 合わせ技術：混合2-3種不同味噌增加風味層次（例：信州70% + 八丁20% + 西京10%）

**餐廳應用**

- 日式定食：附味噌湯為標準配置
- 拉麵店：味噌拉麵湯底（信州味噌+豬骨湯+ニンニク/辣油）
- 居酒屋：味噌燒物（西京燒銀鱈、茄子田樂）、土手鍋

## E. 味醂・清酒——料理的靈魂液體

### 1. 味醂分類與技術

**本味醂（本みりん）**

正統味醂，酒精度12-14%，糖度40-45°Brix。原料：糯米麴+燒酎（蒸餾酒）+糯米，糖化發酵60-90日。製程中麴菌酵素將糯米澱粉分解為葡萄糖、異麥芽糖等多種糖類，形成複雜甜味（非單一蔗糖的單調甜）。酒精抑制雜菌、同時萃取原料風味。

品質指標：
- 糖度：40°Brix以上（高級品達50°Brix）
- 胺基酸：產生鮮味與香氣（麩胺酸200-400mg/100ml）
- 有機酸：檸檬酸、琥珀酸增加風味複雜度

代表品牌：
- 三河みりん（愛知）：天然熟成3年、有機原料
- 福來純（岐阜）：伝統製法、價格¥2,800/700ml
- タカラ本みりん：標準工業產品、價格¥680/1L

**味醂風調味料（みりん風調味料）**

酒精度1%以下（未達酒稅法定義），添加水飴、糖類、調味料製成，無需熟成、成本低廉。甜味單調（主要為葡萄糖漿）、欠缺本味醂的複雜風味與照り（光澤）效果。價格¥200-300/1L，適合大量使用的商業廚房。

**發酵調味料（発酵調味料）**

酒精度8-10%，加鹽2-3%使其不適飲用（避開酒稅），風味介於本味醂與味醂風之間。價格¥400-500/1L，為家庭常用選擇。

**料理效果科學**

1. **照り（光澤）**：糖類於80-100°C加熱產生焦糖化，附著食材表面形成亮澤
2. **消臭**：酒精揮發時帶走魚腥味（胺類化合物）
3. **防崩**：糖類滲入食材產生細胞壁強化，防止煮物碎裂（芋頭、南瓜）
4. **風味**：胺基酸+有機酸+糖類產生多層次甘味
5. **上色**：梅納反應產生金黃至褐色色澤

### 2. 料理用清酒選擇

**清酒 vs 料理酒差異**

| 項目 | 清酒（飲用） | 料理酒 | 料理清酒 |
|------|-------------|--------|----------|
| 精米步合 | 50-70% | 不限（通常90%） | 70-80% |
| 酒精度 | 15-16% | 13-14% | 13-15% |
| 添加鹽 | 無 | 2-3%（避酒稅） | 有/無皆有 |
| 價格 | ¥1,000-5,000/720ml | ¥300-500/1.8L | ¥600-900/1.8L |
| 適用 | 蒸し料理、酒蒸 | 一般煮物 | 高級煮物 |

**選擇建議**

- **普通料理**：料理清酒（月桂冠、白鶴料理用）即可，性價比高
- **蒸し物（酒蒸蛤蜊、鯛魚）**：使用純米酒（無添加釀造酒精），風味清爽
- **燉煮物**：一般清酒，酒精揮發後保留胺基酸鮮味
- **醬汁**：吟釀酒（果香明顯），少量提味

**料理用清酒功能**

1. **嫩化肉質**：酒精溶解蛋白質、有機酸軟化纖維
2. **去腥**：酒精揮發帶走三甲胺（魚腥）
3. **提鮮**：胺基酸含量高（琥珀酸、精胺酸）
4. **增香**：吟釀香、米香融入料理
5. **防腐**：酒精抑制細菌（醃漬物）

**用量標準**

- 魚類煮物：清酒50ml + 醬油30ml + 味醂30ml + 水200ml（4人份）
- 酒蒸：清酒100ml + 水50ml（蛤蜊500g）
- 照燒醬：清酒:醬油:味醂 = 2:2:1
- 醃肉：清酒2大匙 + 薑汁1大匙（豬肉300g，醃30分鐘）

### 3. 澳門採購指南

**本味醂**

- タカラ本みりん（寶酒造）：MOP 48-55/1L，AEON、來來有售
- 福泉本みりん：MOP 65-75/500ml，日本食材店
- 三州三河みりん：MOP 180-220/700ml，專門店特殊訂購

**料理清酒**

- 月桂冠料理清酒：MOP 38-45/1.8L，各大超市
- 白鶴まる：MOP 42-50/1.8L
- タカラ料理のための清酒：MOP 35-42/1.8L

**替代方案（緊急時）**

- 本味醂缺貨 → 發酵調味料（みりんタイプ）+ 少量砂糖
- 料理清酒缺貨 → 飲用清酒（普通酒等級）少量使用
- 完全無酒精需求 → 砂糖+水（無法複製風味，僅提供甜度）

## F. 醋（酢）——酸味的藝術

### 1. 醋的種類與製法

**純米醋（純米酢）**

以白米或糙米釀造，酸度4.2-4.5%，風味圓潤、米香明顯、顏色淡黃透明。製程：米 → 製麴 → 糖化 → 酒精發酵 → 醋酸發酵（Acetobacter菌種將酒精轉化為醋酸，需時30-90日）。高級品使用靜置發酵（表面發酵法），風味較攪拌發酵細緻。

代表品牌：
- ミツカン純米酢：市占率第一、價格¥450/500ml
- 富士酢（京都飯尾釀造）：無農藥米、靜置發酵5個月、價格¥1,200/900ml
- 千鳥酢（村山造酢）：京料理職人指定

**黑醋（黒酢）**

以玄米或大麥釀造，發酵熟成1-3年，顏色深褐、風味濃厚複雜、胺基酸含量為純米醋3-4倍。鹿兒島縣福山町為日本黑醋發祥地，使用陶壺露天發酵（壺內溫差促進菌種代謝）。主要用途為健康飲品、中華料理（酢豚、餃子沾醬）。

- 坂元黑醋（福山）：天然發酵3年、價格¥2,500/700ml
- ミツカン黑醋：工業產品、價格¥680/500ml

**穀物醋（穀物酢）**

混合米、小麥、玉米等穀物釀造，酸度4.5-5%、價格低廉（¥180/900ml）、風味較單薄。適合大量使用（醃漬、清潔）。

**果實醋**

- **柿醋**：和歌山縣特產，柿子自然發酵、酸度3-4%、甘味明顯、富含多酚
- **蘋果醋**：長野縣、青森縣、酸度5%、果香濃郁
- **梅醋**：梅干醃漬時產生的副產品、鹹酸並具、富含礦物質

**酒粕醋（粕酢）**

清酒釀造剩餘酒粕再發酵，風味獨特、價格低、主要用於工業調味料。

### 2. 壽司醋調配技術

**基礎壽司醋配方**

```
材料（1kg米飯用量）：
- 純米醋：200ml
- 砂糖：70-80g（依個人喜好調整）
- 鹽：25g
- 昆布：5cm方形1片

製作：
1. 昆布剪細絲浸入冷醋中，冷藏12小時（提鮮、增加圓潤感）
2. 使用前取出昆布（或保留，江戶前風格會保留昆布絲）
3. 砂糖+鹽加入醋中攪拌至完全溶解
4. 室溫靜置1小時使風味融合
```

**變化配方**

- **江戶前濃醋**：醋220ml + 砂糖60g + 鹽30g（突出酸味）
- **關西甘醋**：醋180ml + 砂糖100g + 鹽20g + 味醂20ml（突出甜味）
- **現代減糖**：醋200ml + 砂糖50g + 鹽25g + 味醂30ml（味醂補償甜度）

**調配科學**

- 糖醋比：1:2.5-2.8為平衡點（過甜掩蓋魚味、過酸刺激）
- 鹽的作用：提鮮（0.8-1%含鹽率）、防腐、增加米粒分離度
- 昆布：釋出麩胺酸（200mg/100ml）、多醣增加滑順感
- 溫度：拌醋時米飯85-90°C、醋室溫（20-25°C）為最佳

### 3. 南蠻漬け基礎

**南蠻漬け定義**

油炸或煎烤的魚肉浸泡醋醬，冷藏保存1-3日使其入味。「南蠻」指室町時代西班牙・葡萄牙傳入的辣椒與醋漬技術。

**基礎南蠻醋配方（4人份）**

```
材料：
- 出汁：200ml
- 醬油：80ml
- 米醋：80ml
- 味醂：60ml
- 砂糖：30g
- 唐辛子：1-2本（切輪片）
- 蔥：1/2根（斜切絲）
- 人參：30g（切絲）

作法：
1. 出汁+醬油+醋+味醂+砂糖煮沸、放涼至室溫
2. 加入唐辛子、蔥絲、人參絲
3. 炸好的魚（小型魚全尾或切塊）趁熱浸入
4. 冷藏4小時至3日
```

**常見食材**

- 小魚：竹筴魚（アジ）、沙丁魚、柳葉魚
- 白身魚：鱸魚、鱈魚切片
- 肉類：雞肉（チキン南蛮）

**技術要點**

- 魚需趁熱（80°C以上）浸入冷醋（溫差使組織膨脹吸收醋醬）
- 醋醬比例：出汁:醬油:醋 = 2.5:1:1為基礎
- 浸泡時間：最短4小時、最佳24小時、最長3日（過久魚肉過酸）

### 4. 其他醋料理應用

**酢の物（醋拌菜）**

```
三杯醋（三杯酢）：醋3:醬油1:味醂1
二杯醋（二杯酢）：醋2:醬油1
甘醋：醋3:砂糖2:鹽少許
土佐醋：三杯醋+柴魚出汁（稀釋用）
```

常見具材：小黃瓜+若布、章魚+海帶芽、蟹肉棒+小黃瓜

**醋飯以外應用**

- **しめ鯖（醋漬鯖魚）**：鮮鯖魚鹽醃1小時 → 純米醋浸泡30分鐘 → 去皮切片
- **ピクルス（日式醃菜）**：甜醋+月桂葉+黑胡椒，醃蘿蔔、人參、小黃瓜
- **酢豚（糖醋肉）**：黑醋:砂糖:醬油:番茄醬 = 3:3:1:1
- **ポン酢**：柑橘汁+醋+醬油+味醂+昆布，用於鍋物、烤魚

**健康效果**

- 血糖控制：醋酸抑制α-澱粉酶，延緩醣類吸收（飯前1大匙醋可降低餐後血糖15-20%）
- 疲勞恢復：檸檬酸參與TCA循環，促進乳酸代謝
- 鈣質吸收：酸性環境促進鈣離子化
- 防腐：pH <4.0抑制大部分細菌生長
